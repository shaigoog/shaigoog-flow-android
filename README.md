# سجل شيقوق — مسار العمليات

مشروع أندرويد جديد بالكامل يعتمد فكرة **ملف العملية**: كل شراء أو بيع يجمع الجهة والبضاعة والوزن والعملة والقيمة في مسار واحد واضح.

الحالة الحالية:
- إنشاء عملية شراء أو بيع.
- عملات متعددة بإجماليات منفصلة.
- وحدات وزن: طن، كيلوجرام، جوال.
- حفظ محلي SQLite.
- لوحة اليوم والعمليات والخزنة.
